RaphEngine

RaphEngine is a game engine that I developed as a personal project. It is written in C++ and uses OpenGL for rendering. The engine is designed to be lightweight and easy to use, making it accessible for beginners while still being powerful enough for more experienced developers.

Features

  • Rendering custom objects
  • Cascaded shadows for higher quality at a lower performance cost
  • LOD (or Level Of Detail) to increase performance
  • RayCasting
  • UI text, image and buttons
  • custom shaders
  • Multi platforms (Linux + Windows)
  • Easy to use

Future features

  • Physics engine
  • Audio engine
  • Networking
  • Particle system
  • Animation system
  • Better documentation
